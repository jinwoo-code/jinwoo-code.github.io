<Button> 태그

Button 태그란?

사용자가 클릭할 수 있는 버튼을 만드는 태그입니다. 가장 기본적이고 많이 사용되는 컨트롤입니다.

<Button Name="myButton" Content="클릭하세요!" Width="100" Height="30" Background="Blue" Foreground="White" Click="Button_Click"/>
주요 속성:
• Content: 버튼에 표시될 텍스트나 내용
• Width/Height: 버튼의 크기
• Background: 배경색
• Foreground: 글자색
• Click: 클릭 이벤트 핸들러
• IsEnabled: 버튼 활성화/비활성화

자식 태그: Button은 단일 자식 요소를 가질 수 있으며, 복잡한 내용을 위해 Grid, StackPanel 등을 자식으로 사용할 수 있습니다.

<TextBox> 태그

TextBox 태그란?

사용자가 텍스트를 입력할 수 있는 입력 필드를 만드는 태그입니다.

<TextBox Name="myTextBox" Text="기본 텍스트" Width="200" Height="25" MaxLength="50" TextWrapping="Wrap"/>
주요 속성:
• Text: 텍스트박스의 내용
• MaxLength: 최대 입력 가능한 글자 수
• TextWrapping: 텍스트 줄바꿈 설정
• AcceptsReturn: Enter 키 입력 허용
• IsReadOnly: 읽기 전용 모드
• PasswordChar: 비밀번호 문자 설정

자식 태그: TextBox는 일반적으로 자식 태그를 가지지 않습니다.

<Label> 태그

Label 태그란?

텍스트를 표시하는 태그로, 다른 컨트롤의 설명이나 제목을 표시할 때 사용합니다.

<Label Content="사용자 이름:" Target="{Binding ElementName=userTextBox}" FontWeight="Bold" Foreground="DarkBlue"/>
주요 속성:
• Content: 레이블에 표시될 내용
• Target: 연결된 컨트롤 지정
• FontWeight: 글자 굵기
• FontSize: 글자 크기
• Foreground: 글자색
• HorizontalAlignment: 수평 정렬

자식 태그: Label은 단일 자식 요소를 가질 수 있으며, 복잡한 내용을 위해 다른 컨트롤들을 포함할 수 있습니다.

<CheckBox> 태그

CheckBox 태그란?

사용자가 선택하거나 해제할 수 있는 체크박스를 만드는 태그입니다.

<CheckBox Name="myCheckBox" Content="동의합니다" IsChecked="True" IsThreeState="False" Checked="CheckBox_Checked"/>
주요 속성:
• Content: 체크박스 옆에 표시될 텍스트
• IsChecked: 체크 상태 (True/False/null)
• IsThreeState: 3상태 모드 (불확정 상태 포함)
• Checked/Unchecked: 체크/해제 이벤트
• IsEnabled: 활성화/비활성화 상태

자식 태그: CheckBox는 Content 속성을 통해 단일 자식 요소를 가질 수 있습니다.

<RadioButton> 태그

RadioButton 태그란?

여러 옵션 중 하나만 선택할 수 있는 라디오 버튼을 만드는 태그입니다.

<RadioButton Name="option1" Content="옵션 1" GroupName="MyGroup" IsChecked="True" Checked="RadioButton_Checked"/>
주요 속성:
• Content: 라디오버튼 옆에 표시될 텍스트
• GroupName: 그룹 이름 (같은 그룹 내에서 하나만 선택)
• IsChecked: 선택 상태
• Checked/Unchecked: 선택/해제 이벤트
• IsEnabled: 활성화/비활성화 상태

자식 태그: RadioButton은 Content 속성을 통해 단일 자식 요소를 가질 수 있습니다.

<ComboBox> 태그

ComboBox 태그란?

드롭다운 목록에서 항목을 선택할 수 있는 콤보박스를 만드는 태그입니다.

<ComboBox Name="myComboBox" Width="150" Height="25" SelectedIndex="0" IsEditable="False"> <ComboBoxItem Content="항목 1"/> <ComboBoxItem Content="항목 2" IsSelected="True"/> <ComboBoxItem Content="항목 3"/> </ComboBox>
주요 속성:
• SelectedIndex: 선택된 항목의 인덱스
• SelectedItem: 선택된 항목
• IsEditable: 편집 가능 여부
• ItemsSource: 데이터 바인딩 소스
• DisplayMemberPath: 표시할 속성 경로

자식 태그: ComboBoxItem 태그들을 자식으로 가집니다.

<ListBox> 태그

ListBox 태그란?

여러 항목을 목록으로 표시하고 선택할 수 있는 리스트박스를 만드는 태그입니다.

<ListBox Name="myListBox" Width="200" Height="120" SelectionMode="Multiple"> <ListBoxItem Content="항목 1"/> <ListBoxItem Content="항목 2" IsSelected="True"/> <ListBoxItem Content="항목 3"/> <ListBoxItem Content="항목 4"/> </ListBox>
주요 속성:
• SelectionMode: 선택 모드 (Single/Multiple/Extended)
• SelectedItems: 선택된 항목들
• ScrollViewer: 스크롤 설정
• ItemsSource: 데이터 바인딩 소스
• DisplayMemberPath: 표시할 속성 경로

자식 태그: ListBoxItem 태그들을 자식으로 가집니다.

<Slider> 태그

Slider 태그란?

사용자가 값을 슬라이더로 조정할 수 있는 컨트롤을 만드는 태그입니다.

<Slider Name="mySlider" Width="200" Height="20" Minimum="0" Maximum="100" Value="50" TickFrequency="10" IsSnapToTickEnabled="True"/>
주요 속성:
• Minimum/Maximum: 최솟값/최댓값
• Value: 현재 값
• TickFrequency: 눈금 간격
• IsSnapToTickEnabled: 눈금에 스냅 여부
• Orientation: 방향 (Horizontal/Vertical)
• SmallChange/LargeChange: 증감 단위

자식 태그: Slider는 일반적으로 자식 태그를 가지지 않습니다.

<ProgressBar> 태그

ProgressBar 태그란?

작업의 진행 상태를 시각적으로 표시하는 프로그레스바를 만드는 태그입니다.

<ProgressBar Name="myProgressBar" Width="200" Height="20" Minimum="0" Maximum="100" Value="65" IsIndeterminate="False"/>
주요 속성:
• Minimum/Maximum: 최솟값/최댓값
• Value: 현재 진행값
• IsIndeterminate: 불확정 모드 (애니메이션)
• Orientation: 방향 (Horizontal/Vertical)
• Foreground: 진행바 색상
• Background: 배경색

자식 태그: ProgressBar는 일반적으로 자식 태그를 가지지 않습니다.

<StackPanel> 태그

StackPanel 태그란?

자식 요소들을 수직 또는 수평으로 순서대로 배치하는 레이아웃 패널입니다.

<StackPanel Orientation="Vertical" Margin="10"> <Button Content="버튼 1" Height="30"/> <Button Content="버튼 2" Height="30"/> <Button Content="버튼 3" Height="30"/> </StackPanel>
주요 속성:
• Orientation: 배치 방향 (Vertical/Horizontal)
• Margin: 외부 여백
• Background: 배경색
• HorizontalAlignment: 수평 정렬
• VerticalAlignment: 수직 정렬

자식 태그: 모든 WPF 컨트롤을 자식으로 가질 수 있습니다.

Grid 태그란?

행과 열로 구성된 테이블 형태의 레이아웃을 만드는 가장 유연한 패널입니다.

<Grid> <Grid.RowDefinitions> <RowDefinition Height="Auto"/> <RowDefinition Height="*"/> </Grid.RowDefinitions> <Grid.ColumnDefinitions> <ColumnDefinition Width="100"/> <ColumnDefinition Width="*"/> </Grid.ColumnDefinitions> <Button Grid.Row="0" Grid.Column="0" Content="버튼"/> <TextBox Grid.Row="0" Grid.Column="1" Text="텍스트박스"/> </Grid>
주요 속성:
• Grid.Row/Column: 자식 요소의 행/열 위치
• Grid.RowSpan/ColumnSpan: 행/열 병합
• RowDefinitions/ColumnDefinitions: 행/열 정의
• Height/Width: Auto, *, 고정값
• ShowGridLines: 격자선 표시 (디버그용)

자식 태그: RowDefinition, ColumnDefinition과 모든 WPF 컨트롤을 자식으로 가집니다.

DockPanel 태그란?

자식 요소들을 상하좌우 가장자리에 도킹하고 남은 공간을 채우는 레이아웃 패널입니다.

<DockPanel LastChildFill="True"> <Menu DockPanel.Dock="Top" Height="25">...</Menu> <StatusBar DockPanel.Dock="Bottom" Height="25">...</StatusBar> <TreeView DockPanel.Dock="Left" Width="200">...</TreeView> <TextBox Name="mainContent">...</TextBox> </DockPanel>
주요 속성:
• DockPanel.Dock: 도킹 위치 (Top/Bottom/Left/Right)
• LastChildFill: 마지막 자식이 남은 공간 채움
• Background: 배경색
• Margin: 외부 여백

자식 태그: 모든 WPF 컨트롤을 자식으로 가질 수 있으며, DockPanel.Dock 속성으로 위치를 지정합니다.

WrapPanel 태그란?

자식 요소들을 순서대로 배치하다가 공간이 부족하면 다음 줄로 자동으로 넘어가는 패널입니다.

<WrapPanel Orientation="Horizontal" ItemHeight="30" ItemWidth="80"> <Button Content="버튼1"/> <Button Content="버튼2"/> <Button Content="버튼3"/> <Button Content="버튼4"/> <Button Content="버튼5"/> </WrapPanel>
주요 속성:
• Orientation: 배치 방향 (Horizontal/Vertical)
• ItemHeight/ItemWidth: 자식 요소의 고정 크기
• HorizontalAlignment: 수평 정렬
• VerticalAlignment: 수직 정렬

자식 태그: 모든 WPF 컨트롤을 자식으로 가질 수 있습니다.

Canvas 태그란?

자식 요소들을 절대 좌표로 정확한 위치에 배치할 수 있는 패널입니다.

<Canvas Width="300" Height="200" Background="LightGray"> <Rectangle Canvas.Left="10" Canvas.Top="10" Width="50" Height="30" Fill="Blue"/> <Ellipse Canvas.Left="100" Canvas.Top="50" Width="40" Height="40" Fill="Red"/> <TextBlock Canvas.Left="200" Canvas.Top="100" Text="Hello" FontSize="16"/> </Canvas>
주요 속성:
• Canvas.Left/Top: 자식 요소의 X, Y 좌표
• Canvas.Right/Bottom: 우하단 기준 좌표
• Canvas.ZIndex: Z축 순서 (겹침 순서)
• ClipToBounds: 경계 자르기 여부

자식 태그: 모든 WPF 컨트롤을 자식으로 가지며, Canvas.Left, Canvas.Top 등으로 위치를 지정합니다.

🖼️ 이미지 영역

Image 태그란?

이미지 파일을 표시하는 태그입니다. 다양한 이미지 포맷을 지원합니다.

<Image Name="myImage" Source="images/photo.jpg" Width="200" Height="150" Stretch="Uniform" HorizontalAlignment="Center"/>
주요 속성:
• Source: 이미지 파일 경로 또는 URI
• Stretch: 크기 조정 방식 (None/Fill/Uniform/UniformToFill)
• Width/Height: 이미지 크기
• Opacity: 투명도 (0.0~1.0)
• RenderTransform: 변환 (회전, 크기 조정 등)

자식 태그: Image는 자식 태그를 가지지 않습니다.

ScrollViewer 태그란?

내용이 컨트롤 크기보다 클 때 스크롤 기능을 제공하는 래퍼 컨트롤입니다.

<ScrollViewer Width="200" Height="150" HorizontalScrollBarVisibility="Auto" VerticalScrollBarVisibility="Auto" CanContentScroll="True"> <TextBlock Text="긴 텍스트 내용..." TextWrapping="Wrap"/> </ScrollViewer>
주요 속성:
• HorizontalScrollBarVisibility: 수평 스크롤바 표시 설정
• VerticalScrollBarVisibility: 수직 스크롤바 표시 설정
• CanContentScroll: 내용 기반 스크롤 여부
• ZoomMode: 줌 기능 활성화
• ScrollChanged: 스크롤 변경 이벤트

자식 태그: 단일 자식 요소를 가지며, 보통 Panel이나 다른 컨트롤을 포함합니다.

<Menu> 태그
파일 편집 보기 도움말

Menu 태그란?

애플리케이션의 메뉴바를 만드는 태그입니다. 계층적 메뉴 구조를 지원합니다.

<Menu Height="25"> <MenuItem Header="파일"> <MenuItem Header="새로 만들기" Click="New_Click"/> <MenuItem Header="열기" Click="Open_Click"/> <Separator/> <MenuItem Header="종료" Click="Exit_Click"/> </MenuItem> <MenuItem Header="편집"> <MenuItem Header="복사"/> <MenuItem Header="붙여넣기"/> </MenuItem> </Menu>
주요 속성:
• MenuItem.Header: 메뉴 항목 제목
• MenuItem.Click: 클릭 이벤트
• MenuItem.IsCheckable: 체크 가능 여부
• MenuItem.Icon: 메뉴 아이콘
• Separator: 구분선 추가

자식 태그: MenuItem 태그들을 자식으로 가지며, 중첩 구조로 서브메뉴를 만들 수 있습니다.

TreeView 태그란?

<TreeView Name="myTreeView" Width="200"> <TreeViewItem Header="루트 항목" IsExpanded="True"> <TreeViewItem Header="자식 항목 1"/> <TreeViewItem Header="자식 항목 2"> <TreeViewItem Header="손자 항목 1"/> <TreeViewItem Header="손자 항목 2"/> </TreeViewItem> </TreeViewItem> </TreeView>

TabControl 태그란?

여러 페이지를 탭으로 구분하여 표시하는 컨트롤입니다. 공간을 효율적으로 사용할 수 있습니다.

<TabControl Name="myTabControl"> <TabItem Header="첫 번째 탭"> <Grid> <TextBlock Text="첫 번째 탭의 내용"/> </Grid> </TabItem> <TabItem Header="두 번째 탭"> <StackPanel> <Button Content="버튼"/> <TextBox Text="텍스트박스"/> </StackPanel> </TabItem> </TabControl>
주요 속성:
• TabItem.Header: 탭 제목
• SelectedIndex: 선택된 탭 인덱스
• SelectedItem: 선택된 탭 항목
• TabStripPlacement: 탭 위치 (Top/Bottom/Left/Right)
• SelectionChanged: 탭 변경 이벤트

자식 태그: TabItem 태그들을 자식으로 가지며, 각 TabItem은 하나의 페이지 내용을 포함합니다.

GroupBox 태그란?

관련된 컨트롤들을 시각적으로 그룹화하고 제목을 표시하는 컨테이너입니다.

<GroupBox Header="사용자 정보" Width="250" Height="150" Margin="10"> <StackPanel Margin="10"> <Label Content="이름:"/> <TextBox Name="nameTextBox"/> <Label Content="이메일:"/> <TextBox Name="emailTextBox"/> </StackPanel> </GroupBox>
주요 속성:
• Header: 그룹박스 제목
• BorderBrush: 테두리 색상
• BorderThickness: 테두리 두께
• FontWeight: 제목 글자 굵기
• Foreground: 제목 글자색

자식 태그: 단일 자식 요소를 가지며, 보통 Panel을 자식으로 하여 여러 컨트롤을 포함합니다.
